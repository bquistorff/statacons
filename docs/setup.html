Installation¶

Install statacons following the installation guide: https://bquistorff.github.io/statacons/installation.html

Files¶

You need to download some files to follow this lesson:

  1. Download statacons-lesson.zip from https://github.com/bquistorff/statacons/blob/main/examples/stataconsIntro.zip.

  2. Move zip into a directory you can access.

  3. Unpack zip.
